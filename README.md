# OutModern_Webform

## Technologies Used

This project was built using the following technologies:

- **Frontend:**
  - HTML + Webforms
  - CSS
  - JavaScript

- **Backend:**
  - WebForms (for server-side logic and dynamic content)
  - MSSQL (Microsoft SQL Server for database management)
